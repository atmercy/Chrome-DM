Getting Started

  1. You will first need to enable API Access to your domain

  2. Have Admin rights / credentials to the domain

    If you are already signed into one or more Google services you may need to sign out to make sure you use the correct credentials

  3. Once the API access is enabled, and you are ready to have an admin sign in go ahead and click on "I'm ready"

Stats

Supporting the following useful metrics:

  • 7 Day Active
  • Total Devices
  • Number of devices on each channel
  • Devices per OU
  • Number of devices on each version

Not Enough?

Download all the information about your devices in a convenient CSV! This includes data that you normally cannot view through the Chrome Management panel.
